## **Typical Contents of /root:**

```
/root/
├── .bashrc                     # Root's bash configuration
├── .profile                    # Root's profile settings
├── .ssh/                       # Root's SSH keys (use with caution!)
│   └── authorized_keys         # SSH keys for root login
├── .cache/                     # Root's cache files
├── .local/                     # Root's local data
│
├── scripts/                    # Administrative scripts
├── backups/                    # System backup files
├── logs/                       # Private log files
│
└── .mysql_history              # MySQL command history (sensitive!)
```
